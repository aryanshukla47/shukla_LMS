# Learning Management System (LMS)

## Overview
This repository contains the source code for a complete Learning Management System, which includes both backend and frontend applications, as well as database configurations.

## Installation
1. Clone the repository.
2. Navigate to the `backend` and `frontend` directories to install dependencies.

## Usage
- Start the backend server with `node server.js`.
- Start the frontend application with `npm start`.

## Structure
- **backend/**: Contains all server-side code.
- **frontend/**: Contains the client-side application.
- **database/**: Contains the database schema.
